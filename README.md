## Description
This is simple static site, which I made while was learning as front-end developer.
Tools, which were used:
- Gulp
- Stylus
- Pug
- Jquery

## Build Setup

``` bash
# install dependencies
npm install

# build project
npm run dev

# build simple server
npm run start

# build for production
npm run deploy
```
